Neural Network: The flowers problem

Idea

Biologist had left some notes about two types of plants: with red flowers and with blue flowers. His last entry was left without flower color. This neural network tries to guess the last flower color using biologist data about the rest of plants.

Data

  • Length, width, color
  • 1, 1, blue
  • 2, 1, blue
  • 2, 0.5, blue
  • 3, 1, blue
  • 3, 1.5, red
  • 3.5, 0.5, red
  • 4, 1.5, red
  • 5.5, 1, red
  • 4.5, 1, ???
